HashSet实现原理

特点

  1. 基于HashMap实现
  2. 无序
  3. 允许null值
  4. 元素不重复
  5. 非线程安全

数据结构和初始化

HashSet底层是基于HashMap实现的，它的所有数据都存放到HashMap中，更准确的说应该的HashMap的key中，所以无法保证有序，但保证元素的唯一性(HashMap的key是唯一的，本质还是通过key对象的equals和hashCode方法实现的)，而HashMap的value值则直接赋值PRESENT常量。
